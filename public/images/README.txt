Hero Background Image

This directory contains images used in the hero section of the website.

To use a different hero background image:
1. Place your desired image in this directory
2. Name it "hero-background.jpg" to automatically use it with the current configuration
3. Or update the HeroSection.jsx file with the new image path

Best practices:
- Use high-quality images (recommended min resolution: 1920x1080)
- Use images with good contrast to ensure text readability
- Optimize images for web to keep page load times fast
